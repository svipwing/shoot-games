# 射击小游戏

![预览图](image/help.png)

## 介绍

一款休闲类的射击小游戏

## 玩法

移动端：
1. 点击“向左”和“向右”来控制“武器”移动
2. 点击“开枪射击”来射击

电脑端：
1. 按下键盘的“A”和“D”来控制“武器”移动
2. 按下键盘的“S”来射击

## 部署

1. 下载本项目的发行版（zip压缩包）
2. 把zip解压到 服务器/虚拟主机（您也可以选择解压到本地运行）
3. 访问`index.html`，开始游玩
